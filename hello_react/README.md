## Hello React

### Component는 클래스형과 함수형이 있다.

- 클래스형

  > class Component extends Component 문법으로 선언하고  
  > render() { return ( jsx 코드 ) } 형식이다

- 함수형

  > const()=>{ return (jsx 코드 ) }

- Componenet 연결

  > 부모 Component에서 수행  
  > import 객체이름 from "파일위치"  
  > <객체이름 />

  > 변수 값을 전달하며 사용하기 위해선
  > <객체이름 변수명="값 />
